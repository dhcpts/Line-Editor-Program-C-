# Line Editor Program (C++)

An organized multi-file implementation of a line editor program, supporting file reading, writing and modification via a buffer.

## Overview
This program implements various functions to allow access to text inside documents. Commands for the editor include:
  - Reading from a specified document, or from a currently saved default document if a new one was created.
  - Writing to/creating a document to output the buffer text into while saving changes to the text.
  - Inserting text at the current line the user is at in the buffer.
  - Inserting text after the current line the user is at in the buffer.
  - Jump command to move through text lines in the buffer.
  - Delete command to delete a single line, or a range of lines.
  - List command to list one line, or a range of lines.
  - Help command to list out commands.

## Tech Stack
  - Language: C++
  - IDE: Visual Studio
